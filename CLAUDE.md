# Claude Code Documentation

This project uses [Claude Code](https://code.claude.com/docs/) - Anthropic's official CLI for Claude.

## About Claude Code

Claude Code is an interactive CLI tool that helps with software engineering tasks. It provides:

- **Code Assistance**: Bug fixing, feature implementation, code refactoring
- **File Operations**: Reading, writing, and editing files
- **Command Execution**: Running bash commands and scripts
- **Project Navigation**: Searching and exploring codebases
- **Version Control**: Git operations and repository management
- **Web Integration**: Searching and fetching web content

## Key Features

- Multi-tool parallel execution for efficiency
- Task management with todo lists
- Interactive user questions during execution
- Background process management
- Code security awareness (OWASP top 10)

## Usage

```bash
claude-code
```

### Common Commands

- `/help` - Get help with using Claude Code
- `/clear` - Clear the conversation
- `/tasks` - View running background tasks
